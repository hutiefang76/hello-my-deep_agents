# Sam Altman · OpenAI CEO · 2025-2027 Agent Roadmap

> OpenAI CEO · YC 前总裁 · "AGI 营销之神"
> 影响力: 他放话的时间节点会成为整个行业的预期锚点

---

## 1. 核心论断: 三年路线图 (2025 / 2026 / 2027)

> *"In 2025, we may see the first AI agents 'join the workforce' and materially change the output of companies."*
> *"2026 will likely see the arrival of systems that can figure out novel insights."*
> *"2027 may see the arrival of robots that can do tasks in the real world."*

**中文**:
- **2025**: AI Agent 加入劳动力，**实质改变公司产出**
- **2026**: 出现能找到**新洞见**的系统
- **2027**: 能在物理世界完成任务的机器人

### 时间线含义

```
2025 ─────── Agent 元年
            │ 客服 / 个人助理 / 代码助手 / 数据分析助手
            │ ← 我们这本教程目标定位
            ▼
2026 ─────── Discovery 元年
            │ 新洞见 / 科研发现 / 设计创新
            ▼
2027 ─────── Embodied 元年
            │ 机器人 / 物理操作
```

---

## 2. AGI 信心论断

> *"We're now confident we know how to build AGI as we have traditionally understood it."*

**中文**: 我们现在确信知道怎么按传统定义构建 AGI。

(注: 这里他偷换了 AGI 的定义——OpenAI 内部把 AGI 重定义为"经济价值产出超过中位数员工"，**不是**人类智能等价。但这是营销策略，不是技术承诺。)
